# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-13

### Added

- Initial release of Scribi
- Core editor components (`EditorRoot`, `EditorContent`)
- Bubble menu for text selection formatting
- Command palette with slash commands
- Customizable image upload system (no vendor lock-in)
- Rich text extensions (bold, italic, underline, etc.)
- Code blocks with syntax highlighting
- Task lists with checkboxes
- Markdown support
- Link and YouTube embed support
- Character count tracking
- Full TypeScript support
- React 19 compatibility
- Comprehensive documentation

### Philosophy

- Developer-first design
- No analytics or tracking
- No forced AI features
- Complete customization
- Framework-agnostic storage
